
Install Python
Open a Terminal 
Change in the folder of this tool /src/
---
> Install the requirements
pip install -r requirements.txt
depending on ur system the command can also be
--> py -m pip install -r requirements.txt
--> python3 -m pip install -r requirements.txt
---
> Run the tool
py main.py
depending on ur system the command can also be
-> python3 main.py
-> python main.py
